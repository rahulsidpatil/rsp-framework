# Basic Workflows

## Overview

Simple examples demonstrating core RSP Framework concepts.

## Available Examples

### [Hello World](hello-world/)
Simplest possible example showing structured prompting

**Topics Covered**:
- Basic prompt structure
- Simple code generation
- Testing validation

**Time**: 15 minutes

---

### [REST API](rest-api/)
Building a basic REST API with AI assistance

**Topics Covered**:
- API design with AI
- Code generation for endpoints
- Test generation
- Security basics

**Time**: 1 hour

---

### [CRUD App](crud-app/)
Complete CRUD application example

**Topics Covered**:
- Requirements extraction
- Database design
- Frontend + Backend development
- Integration testing

**Time**: 2-3 hours

---

## Learning Path

1. Start with **Hello World** to understand basics
2. Move to **REST API** for practical application
3. Complete **CRUD App** for full workflow

## What You'll Learn

- How to structure prompts
- How to manage context
- How to validate AI outputs
- How to iterate effectively
- How to document work
