# Utility Scripts

## Overview

Helper scripts for RSP Framework workflows.

## Available Scripts

### validate-prompts.sh
Validate prompt file structure

```bash
./validate-prompts.sh prompts/
```

Checks:
- File naming conventions
- Required sections present
- Valid markdown format

---

### generate-docs.sh *(Placeholder)*

```bash
./generate-docs.sh src/
```

## Usage

1. Make scripts executable: `chmod +x script-name.sh`
2. Run from project root
3. Review output
