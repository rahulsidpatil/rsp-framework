# Tools

## Overview

Tooling to support RSP Framework usage.

## Available Tools

### [RSP CLI](rsp-cli/README.md) *(Future)*
Command-line tool for RSP Framework workflows

**Features** (planned):
- Initialize RSP structure in projects
- Generate prompts from templates
- Validate prompt structure
- Track metrics

### [Scripts](scripts/README.md)
Utility scripts for common tasks

**Available Scripts**:
- `validate-prompts.sh` - Validate prompt structure
- `generate-docs.sh` - Generate documentation
- More to come...
