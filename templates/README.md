## Template Conventions

### Placeholders
- `[PLACEHOLDER]` - Replace with your content
- `[OPTIONAL]` - Include if applicable
- `<!-- Comment -->` - Guidance notes

### Format
- All templates in Markdown
- Self-contained and copyable
- Include usage examples
- Provide guidance
