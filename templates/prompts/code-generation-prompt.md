# Code Generation Prompt Template

See detailed pattern at: [Code Generation Pattern](../../docs/03-patterns/prompt-patterns/code-generation.md)

## Quick Template

```markdown
# Implement: [Component/Feature Name]

## Project Context
- Language: [Programming language]
- Framework: [Framework/library]
- Architecture: [Pattern]
- Style: [Conventions]

## Requirements
[From requirements extraction or clear specifications]

## Technical Specifications
- **Input/Output**: [What it receives/returns]
- **Interface/API**: [Signatures, endpoints, props]
- **Data Models**: [Structures]
- **Dependencies**: [Libraries, modules]

## Quality Requirements
- **Code Style**: [Style guide, linter]
- **Error Handling**: [Requirements]
- **Security**: [Requirements]
- **Performance**: [Targets]
- **Testing**: [Coverage goals, scenarios]

## Please provide:
1. Implementation code
2. Inline documentation
3. Error handling
4. Input validation
5. Unit tests
6. Usage examples
```

For full details and examples, see the [Code Generation Pattern](../../docs/03-patterns/prompt-patterns/code-generation.md).
