# Refactoring Prompt Template

See detailed pattern at: [Refactoring Pattern](../../docs/03-patterns/prompt-patterns/refactoring.md)

## Quick Template

```markdown
# Refactor: [Component/Module Name]

## Current Code
```[language]
[Paste code to refactor]
```

## Context
- **Purpose**: [What this code does]
- **Usage**: [How it's used]
- **Constraints**: [What can't change]

## Issues Identified
1. [Issue 1]
2. [Issue 2]
3. [Issue 3]

## Refactoring Goals
- [ ] [Goal 1]
- [ ] [Goal 2]
- [ ] [Goal 3]

## Requirements
- Preserve existing functionality
- Maintain backward compatibility
- Follow [style guide]
- Improve test coverage to [X%]

## Please provide:
1. Analysis of current issues
2. Proposed refactoring approach
3. Refactored code
4. Explanation of changes
5. Updated/new tests
6. Migration notes (if needed)
```

For full details and examples, see the [Refactoring Pattern](../../docs/03-patterns/prompt-patterns/refactoring.md).
