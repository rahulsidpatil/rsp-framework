# Bug Fix Workflow

Systematic approach to debugging and fixing issues with AI assistance.

## Quick Workflow

### Step 1: Reproduce (15-30 min)
- [ ] Gather bug report details
- [ ] Reproduce locally
- [ ] Document steps to reproduce
- [ ] Identify affected components

### Step 2: Root Cause Analysis (30-60 min)
- [ ] Review related code
- [ ] Check logs and errors
- [ ] Test hypotheses
- [ ] Identify root cause
- [ ] Use AI for analysis

### Step 3: Solution Design (15-30 min)
- [ ] Design fix
- [ ] Consider side effects
- [ ] Plan testing approach
- [ ] Document decision

### Step 4: Implementation (30-60 min)
- [ ] Implement fix
- [ ] Add regression test
- [ ] Verify fix works
- [ ] Check for side effects

### Step 5: Verification (15-30 min)
- [ ] Test original reproduction steps
- [ ] Run full test suite
- [ ] Check related functionality
- [ ] Document fix
