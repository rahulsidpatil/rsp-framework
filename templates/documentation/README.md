# Documentation Templates

## Overview

Templates for documenting AI-assisted development.

## Available Templates

### [Decision Record (ADR)](decision-record.md)
Architecture Decision Record template for AI-related decisions

### [Prompt Changelog](prompt-changelog.md)
Track evolution of prompts over time

### [Context Document](context-document.md)
Persistent context for AI interactions

## How to Use

1. Choose appropriate template
2. Fill in details
3. Store in version control
4. Update as needed
5. Reference in prompts
