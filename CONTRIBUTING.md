# Contributing to RSP Framework

Thank you for your interest in contributing to the RSP Framework! This document provides guidelines for contributing to the project.

## How to Contribute

Contributions are welcome in many forms:

- **Documentation improvements**: Help improve or expand the documentation
- **Examples**: Share real-world examples and case studies
- **Templates**: Contribute reusable prompt templates and workflows
- **Bug reports**: Report issues you encounter
- **Feature requests**: Suggest new features or improvements

## Getting Started

1. Fork the repository
2. Create a new branch for your contribution
3. Make your changes
4. Submit a pull request

## Code of Conduct

Please be respectful and constructive in all interactions.

## Questions?

Feel free to open an issue for questions or discussion.
