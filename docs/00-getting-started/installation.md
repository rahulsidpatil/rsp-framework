# Installation

## Prerequisites

RSP Framework is a methodology and set of practices. No specific installation is required, but you'll need:

- An AI coding assistant (e.g., Claude, Cursor, GitHub Copilot, etc.)
- Version control system (Git recommended)
- Your preferred development environment

## Setting Up Your Environment

### 1. Choose Your AI Tool

Select an AI coding assistant that fits your workflow. RSP Framework is tool-agnostic.

### 2. Clone RSP Framework Templates

```bash
git clone https://github.com/rahulsidpatil/rsp-framework.git
```

### 3. Explore Templates

Navigate to the `templates/` directory to find:
- Prompt templates
- Workflow templates
- Documentation templates
- Checklists

### 4. Integrate into Your Workflow

Copy relevant templates to your project and adapt them to your needs.
