# Tools Reference

## Overview

Quick reference for tools mentioned in RSP Framework.

See [Tooling Ecosystem](../02-framework/pragmatic/tooling.md) for comprehensive guide.

## AI Coding Assistants

- **Claude Code / Claude**: Advanced reasoning, large context
- **GitHub Copilot**: IDE integration, real-time suggestions
- **Cursor**: AI-native IDE, multi-file editing
- **Codeium**: Fast autocomplete, free tier
- **Tabnine**: Enterprise features, on-premise options

## Version Control

- **Git**: Essential version control
- **GitHub / GitLab / Bitbucket**: Collaboration platforms

## Testing

- **Jest**: JavaScript testing
- **pytest**: Python testing
- **go test**: Go testing
- **Cypress**: E2E testing

## Code Quality

- **ESLint**: JavaScript linting
- **Prettier**: Code formatting
- **SonarQube**: Code quality platform
- **pylint**: Python linting
- **golangci-lint**: Go linting

## Security

- **Snyk**: Security scanning
- **npm audit / yarn audit**: Dependency security
- **Bandit**: Python security
- **gosec**: Go security
